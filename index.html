How It Works

Translate mobile input into mouse or keyboard actions

MobControl acts as a virtual input layer. Send movement commands, simulate mouse clicks, or trigger keystrokes — all mapped from your mobile interface.

Core Features at a Glance

MobControl is secure, modular, and easy to use

QR Pairing

Pair your phone and desktop securely using a temporary session token and key fingerprint.

Custom UI

Design your own controller layout and configure gestures, buttons, or accelerometer zones.

Works Offline

No internet connection required; functions entirely over local WiFi/LAN.

Universal Support

Works with any game that supports keyboard/mouse input – no game-side changes needed.

Open Source

View the source code and contribute on GitHub. Built with Unity, WPF, and modern web tools.

Real-time Sync

High-speed syncing of inputs with low latency via WebSocket or TCP mode.
