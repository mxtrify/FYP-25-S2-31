Secure QR Code Pairing

Fast, safe, and easy device connection

MobControl uses QR code-based pairing with pre-shared keys and TLS encryption to establish secure communication between your phone and desktop.

Each session token expires after a short period, and key fingerprints are visually verified to avoid man-in-the-middle attacks.

Next

Custom Layouts & Input Mapping

Design your own controller with drag-and-drop editing

Users can assign touch zones, gestures, accelerometer input, and button mappings. Layouts are stored in JSON for flexibility and shared use.

Core Features at a Glance

MobControl is secure, modular, and easy to use

QR Pairing

Pair your phone and desktop securely using a temporary session token and key fingerprint.

Custom UI

Design your own controller layout and configure gestures, buttons, or accelerometer zones.

Encrypted Control

All communication is encrypted over TLS to keep your control data private and reliable.

Universal Support

Works with any game that supports keyboard/mouse input – no game-side changes needed.

Open Source

View the source code and contribute on GitHub. Built with Unity, WPF, and modern web tools.

Real-time Sync

High-speed syncing of inputs with low latency via WebSocket or TCP mode.
